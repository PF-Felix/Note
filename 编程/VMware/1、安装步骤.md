# 问题与解决

以前的华为笔记本，装虚拟机用的是**VMware 15.5**+**CentOS-7-Minimal-1810.iso**

此次换了零刻小主机之后，还按华为笔记本的方式安装虚拟机发现虚拟机安装失败了，报错如下：

```shell
Kernel panic - not syncing: Fatal exception
```

折腾了许久，最终的解决方式是用**VMware 16**+**CentOS-7-x86_64-Minimal-2003.iso**

**安装必备软件**

```shell
yum -y install net-tools
yum -y install wget
yum -y install vim
```

**同步时钟**

同步时钟可以配置阿里云的服务器
可以配置多个时间同步服务器来提高时间同步的可靠性和准确性

```shell
yum -y install ntp
vi /etc/ntp.conf

server ntp1.aliyun.com
server ntp2.aliyun.com
server ntp3.aliyun.com
server ntp4.aliyun.com

service ntpd start
chkconfig ntpd on
```

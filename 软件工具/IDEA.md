# 快捷键

- **搜索**
    - 全文搜索：ctrl + shift + F
    - 文件搜索：shift + shift
- **常用**
    - 自动生成方法：alt + insert
    - 删除行/复制行：ctrl + Y/D
    - 变量自动补全：ctrl + alt + v
    - 字符串大小写转换：ctrl + shift + u
- **其他**
    - 查看继承关系：ctrl + H
    - 重命名：shift + F6
    - 啦啦啦：sout、main、fori、iter
